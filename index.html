Western Ferries Status App

The Western Ferries Status App is a simple community tool that helps commuters quickly check the current status of the Dunoon to Gourock ferry service.

About the app

This project was created independently to make it easier for people travelling between Dunoon and Gourock to quickly see whether the ferry service is running normally, partially disrupted, or suspended.

The app is not affiliated with Western Ferries Ltd. It simply provides a convenient way to monitor publicly available service information.
